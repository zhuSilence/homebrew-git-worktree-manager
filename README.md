# Homebrew Tap for Git Worktree Manager

A GUI tool for managing Git worktrees.

## Installation

```bash
# Add the tap
brew tap zhuSilence/git-worktree-manager

# Install the cask
brew install --cask git-worktree-manager
```

## Usage

After installation, you can:

1. Open **Git Worktree Manager** from your Applications folder
2. Run `git-worktree-manager` from the command line

## Updates

```bash
brew upgrade git-worktree-manager
```
